Service gouvernemental en ligne 1

Ce service vous permet de soumettre un demande au service gouvernemental 1. Vous devrez remplir les champs d'un formulaire et le transmettre. Vous receverez ensuite une réponse dans un court délais.

Pour utiliser ce service, vous aurez besoin de votre : 

  • Numéro d’entreprise du Québec (NEQ) à 10 chiffres
  • Code d’accès (8 lettres et chiffres)

Si vous n'avez pas de code d’accès, bonne chance pour en obtenir un. En effet, un tel code est si secret que personne ne sait vraiment comment l'obtenir.
